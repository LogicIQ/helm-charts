1. Check the status of the pvc-chonker controller:
   kubectl get pods -n {{ .Release.Namespace }} -l "{{ include "pvc-chonker.selectorLabels" . }}"

2. View the logs:
   kubectl logs -n {{ .Release.Namespace }} -l "{{ include "pvc-chonker.selectorLabels" . }}" -f

3. Access metrics endpoint:
   kubectl port-forward -n {{ .Release.Namespace }} svc/{{ include "pvc-chonker.fullname" . }}-controller 8080:8080
   # Then visit http://localhost:8080/metrics

4. To enable auto-expansion on a PVC, add these annotations:
   kubectl annotate pvc <pvc-name> pvc-chonker.io/enabled=true
   kubectl annotate pvc <pvc-name> pvc-chonker.io/threshold=80%
   kubectl annotate pvc <pvc-name> pvc-chonker.io/increase=10%

For more information on configuration options, see:
https://github.com/logicIQ/pvc-chonker
